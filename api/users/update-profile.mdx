title: Update profile

---

## PUT /api/auth/update-profile

Updates the authenticated user's profile details. If a profile picture is provided, it will be uploaded to Cloudinary.

### Authentication

This endpoint requires authentication via JWT token in the Authorization header.

### Request body

<ParamField name="fullName" type="string">
  User's full name
</ParamField>

<ParamField name="bio" type="string">
  User's biography or status message
</ParamField>

<ParamField name="profilePic" type="string">
  Base64 encoded image data for profile picture. If not provided, only name and bio will be updated.
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates whether the update was successful
</ResponseField>

<ResponseField name="user" type="object" required>
  Updated user object
  
  <ResponseField name="_id" type="string">
    User's unique identifier
  </ResponseField>
  
  <ResponseField name="fullName" type="string">
    Updated full name
  </ResponseField>
  
  <ResponseField name="email" type="string">
    User's email address
  </ResponseField>
  
  <ResponseField name="bio" type="string">
    Updated biography
  </ResponseField>
  
  <ResponseField name="profilePic" type="string">
    URL to uploaded profile picture (Cloudinary URL)
  </ResponseField>
</ResponseField>

### Example request

```bash
curl -X PUT https://api.quickchat.com/api/auth/update-profile \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "fullName": "John Doe Updated",
    "bio": "Senior Software Engineer",
    "profilePic": "data:image/jpeg;base64,/9j/4AAQSkZJRg..."
  }'
```

### Example response

```json
{
  "success": true,
  "user": {
    "_id": "60d5ec49f1b2c72b8c8e4f1a",
    "fullName": "John Doe Updated",
    "email": "john@example.com",
    "bio": "Senior Software Engineer",
    "profilePic": "https://res.cloudinary.com/demo/image/upload/v1234567890/profile.jpg"
  }
}
```

### Error response

```json
{
  "success": false,
  "message": "Error message description"
}
```
