## Sharing images in chats

Send images to any user in your conversations:

<Steps>
  <Step title="Open a conversation">
    Select a user from the sidebar to open their chat.
  </Step>
  
  <Step title="Click the gallery icon">
    Find the gallery icon next to the message input field at the bottom.
  </Step>
  
  <Step title="Select your image">
    Choose a PNG or JPEG image from your device.
  </Step>
  
  <Step title="Image sent automatically">
    The image uploads and sends immediately - no additional confirmation needed!
  </Step>
</Steps>

### Supported image formats

QuickChat accepts the following image types:

- **PNG**: .png files
- **JPEG**: .jpg and .jpeg files

<Note>
If you select a file that's not an image, you'll receive an error notification asking you to choose a valid image format.
</Note>

## How image upload works

When you share an image, QuickChat:

1. **Converts to base64**: The image is encoded in your browser
2. **Uploads to cloud**: Sent securely to Cloudinary for storage
3. **Generates URL**: A permanent, secure URL is created
4. **Saves to database**: The URL is stored with your message
5. **Delivers instantly**: Sent in real-time to the recipient

## Image display in chats

When images are shared in conversations:

### Image messages

- **Full display**: Images appear in the chat at up to 230px width
- **Border styling**: A subtle border distinguishes images from text
- **Rounded corners**: Images have rounded corners for a modern look
- **Preserved aspect ratio**: Images maintain their original proportions

### Layout behavior

- **Your images**: Appear on the right side of the chat
- **Their images**: Appear on the left side
- **With avatar**: Small profile picture shows next to each image
- **With timestamp**: Time sent is displayed below the image

<Tip>
Images automatically scroll into view when sent or received, so you never miss new media!
</Tip>

- **Display**: Shown as a circular avatar
- **Locations**: Appears in sidebar, chat headers, and messages

## File size and limits

QuickChat has sensible limits for image uploads:

### Upload limits

- **Maximum request size**: 4MB total
- **Recommended**: Keep images under 2MB for optimal performance
- **No file count limit**: Share as many images as you want

<Note>
The 4MB limit applies to the entire request, which includes the base64-encoded image. Very large images may be rejected.
</Note>

1. **Instant upload**: Image is uploaded to Cloudinary immediately

3. **Live display**: Recipient sees the image instantly if online

## Troubleshooting image uploads

### Common issues and solutions

**"Please select an image" error**
- Ensure you're selecting a PNG or JPEG file
- Check that the file extension is .png, .jpg, or .jpeg

**Image not uploading**
- Check your internet connection
- Verify the file size is under 4MB
- Try a different image format

**Image appears pixelated**
- Use higher resolution source images
- Avoid uploading already-compressed images

<Note>
If you continue experiencing issues with image uploads, check your network connection and ensure you're using a supported browser.
</Note>

## Image persistence

All uploaded images are permanently stored:

- **Chat history**: Images remain in conversation history indefinitely
- **Profile pictures**: Stay until you update them
- **No expiration**: Images don't expire or get deleted automatically
- **Cloud backup**: Stored reliably in Cloudinary's infrastructure
