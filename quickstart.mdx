Get QuickChat up and running on your local machine and send your first real-time message in minutes.

<Steps>
  <Step title="Clone the repository">
    Clone the QuickChat repository to your local machine:

    ```bash
    git clone https://github.com/your-username/quickchat.git
    cd quickchat
    ```
  </Step>

  <Step title="Set up the server">
    Navigate to the server directory and install dependencies:

    ```bash
    cd server
    npm install
    ```

    <Note>
      QuickChat uses Node.js with Express, Socket.IO for real-time messaging, and MongoDB for data storage.
    </Note>

    Create a `.env` file in the `server` directory with the following environment variables:

    ```bash
    MONDODB_URI=your_mongodb_connection_string
    JWT_SECRET=your_jwt_secret_key
    PORT=5000
    NODE_ENV=development
    CLOUDINARY_CLOUD_NAME=your_cloudinary_cloud_name
    CLOUDINARY_API_KEY=your_cloudinary_api_key
    CLOUDINARY_API_SECRET=your_cloudinary_api_secret
    ```

    <Tip>
      You can use [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) for a free cloud database, and [Cloudinary](https://cloudinary.com/) for image uploads (profile pictures and message images).
    </Tip>
  </Step>

  <Step title="Set up the client">
    Open a new terminal window, navigate to the client directory, and install dependencies:

    ```bash
    cd client
    npm install
    ```

    The client is built with React, Vite, and TailwindCSS for a modern, responsive UI.
  </Step>

  <Step title="Start the application">
    You'll need two terminal windows to run both the server and client.

    <CodeGroup>
      ```bash Server
      cd server
      npm start
      ```

      ```bash Client
      cd client
      npm run dev
      ```
    </CodeGroup>

    The server will start on `http://localhost:5000` and the client will start on `http://localhost:5173` (default Vite port).

    <Warning>
      Make sure MongoDB is connected before the server starts. Check the terminal for "Database Connected" confirmation.
    </Warning>
  </Step>

  <Step title="Send your first message">
    Now you're ready to send your first message:

    1. Open your browser and navigate to `http://localhost:5173`
    2. **Create an account** by clicking the sign-up link and filling in your details (full name, email, password, and bio)
    3. **Create a second account** in an incognito/private browser window or different browser
    4. **Select a user** from the sidebar to start a conversation
    5. **Type a message** and hit send

    <Tip>
      Try sending messages from both accounts to see real-time updates powered by Socket.IO. Messages appear instantly without page refresh!
    </Tip>

    Your message will be sent via the WebSocket connection and stored in MongoDB. The server-side code handling this is in `server/controllers/msgController.js:71-101`:

    ```javascript server/controllers/msgController.js
    export const sendMessage = async (req, res) => {
      try {
        const { text, image } = req.body;
        const receiverId = req.params.id;
        const senderId = req.user._id;

        let imageUrl;
        if (image) {
          const uploadResponse = await cloudinary.uploader.upload(image);
          imageUrl = uploadResponse.secure_url;
        }

        const newMessage = await Message.create({
          senderId,
          receiverId,
          text,
          image: imageUrl,
        });

        //Emit the new message to the receiver's socket
        const receiverSocketId = userSocketmap[receiverId];
        if (receiverSocketId) {
          io.to(receiverSocketId).emit("newMessage", newMessage);
        }

        res.json({ success: true, newMessage });
      } catch (error) {
        console.log(error.message);
        res.json({ success: false, message: error.message });
      }
    };
    ```
  </Step>
</Steps>
