## What is QuickChat?

QuickChat is a modern, real-time web chat application built with React and Node.js. It allows users to send and receive messages instantly using WebSocket connections powered by Socket.IO.

## Key features

<AccordionGroup>
  <Accordion title="Real-time communication">
    QuickChat uses Socket.IO to establish WebSocket connections between clients and the server, enabling instant message delivery without polling. Users see messages appear immediately as they're sent.
  </Accordion>

  <Accordion title="User profiles">
    Each user has a customizable profile with a full name, bio, and profile picture. Update your profile anytime from the profile page.
  </Accordion>

  <Accordion title="Message status tracking">
    QuickChat tracks message delivery and read status, so you know when your messages have been seen by recipients.
  </Accordion>

  <Accordion title="Responsive design">
    Built with Tailwind CSS and React, QuickChat provides a beautiful, responsive interface that works seamlessly across desktop and mobile devices.
  </Accordion>
</AccordionGroup>

## Tech stack

QuickChat is built with modern web technologies:

<CardGroup cols={2}>
  <Card title="Frontend" icon="react">

    - Socket.IO Client 4.8.1
    - React Router DOM 7.6.0
    - Axios for API requests
  </Card>
  <Card title="Backend" icon="server">
    - Node.js with Express 5.1.0
    - Socket.IO 4.8.1

</CardGroup>
